# 🌐 Web Development Lab

A collection of frontend and backend web development experiments created as part of my college Web Development Lab coursework.

This repository contains projects built using HTML, CSS, JavaScript, Node.js, Express.js, MongoDB, and Mongoose.

## 🔹 Book API (Express.js)

REST API project built using Node.js and Express.js.

### Features
- Get all books
- Add new books
- Update existing books
- Delete books

### API Methods

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/books` | Fetch all books |
| POST | `/books` | Add a new book |
| PUT | `/books/:id` | Update a book |
| DELETE | `/books/:id` | Delete a book |

---


---
